Fonctionnalités CSS Moderne

Découvrez les puissantes fonctionnalités qui ont révolutionné le développement web

CSS Grid

Système de mise en page bidimensionnel pour créer des designs complexes et responsives avec peu de code.

Flexbox

Modèle de mise en page unidimensionnel pour aligner et distribuer l'espace entre les éléments.

Variables CSS

Créez des valeurs réutilisables pour maintenir la cohérence et faciliter les mises à jour.

Animations & Transitions

Animez vos interfaces sans JavaScript avec des transitions fluides et des keyframes.

Media Queries

Exemples de Code

Apprenez avec des exemples pratiques que vous pouvez tester immédiatement

Variables CSS
/* Déclarer des variables globales */
:root {
  --primary-color: #4361ee;
  --spacing: 1rem;
  --border-radius: 8px;
}

/* Utiliser les variables */
.button {
  background: var(--primary-color);
  padding: var(--spacing);
  border-radius: var(--border-radius);
}
CSS Grid
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}

/* Pour les écrans mobiles */
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
Flexbox
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
}

.logo {
  margin-right: auto;
}

.nav-links {
  display: flex;
  gap: 1.5rem;
}
